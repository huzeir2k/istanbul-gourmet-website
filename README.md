# Istanbul Gourmet Market

![Istanbul Gourmet Logo](./public/istanbul-gourmet-logo.png)

---

## Overview

This project is a **Laravel + Vue.js application** that combines powerful backend capabilities with a modern, dynamic frontend. It leverages Laravel's robust routing, authentication, and database features, while Vue.js provides a reactive user interface for an enhanced user experience.

---

## About Istanbul Gourmet

**Istanbul Gourmet Market** is a specialty store offering a curated selection of high-quality products. The market features imported goods directly from Turkey alongside locally sourced items, delivering a unique mix of authenticity and freshness.

The brand emphasizes tradition, quality, and a connection to Turkish culture while appealing to the local community.

---

## Tech Stack

- **Backend**: Laravel (PHP framework)  
- **Frontend**: Vue.js (progressive JavaScript framework)  
- **Bundler**: Vite  
- **Styling**: Tailwind CSS (optional depending on customization)  

---

## Features

- Full Laravel backend with authentication, routing, and API endpoints  
- Vue.js integration for modern reactive UI components  
- Scalable architecture for adding new features and pages  
- Extensible design suitable for both e-commerce and content-driven functionality
